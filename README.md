# gt-burger-joint

  ## Description

  Eat-Da-Burger! is a restaurant app that lets users input the names of burgers they'd like to eat. When a user submits burger's name the app will then let the user click a button to devour it and move it to the eaten column.

  ## Installation

  To install necessary dependencies, run <code>npm install</code>.<br> 
  Dependencies: MySQL Express Handlebars

  ## Usage
  I have provided a schema and seeds for you to input into the database. You will need to put in your own password into the connection.js file. you can access the deployed version on [Heroku](https://benefique-livre-39667.herokuapp.com/) as well.
